# Production Code Functions
## CO2 Production Code
### load_co2_rows()
**Purpose:** The function loads the CO2 CSV dataset, filters out rows with missing CO2 values, converts each row into a CO2row dataclass, and returns the list.
### entities()
**Purpose:** The function returns a unique list of entity names from the CO2 rows,which may be optionally restricted to a set of countries.
